# fisheye-tool - Claude Code 開発ガイド

## バージョン管理ルール
- **版の唯一の源**: `index.html` 先頭付近の `const APP_VERSION = 'vX.Y.Z';` のみ。
  重複定義・HTML内の版リテラルの直書き禁止。画面表示・バッジ追記はすべてこの値を参照する。
- タスク完了ごとに SemVer でバンプしてからコミット:
  - 破壊的変更・大機能: メジャー (x.0.0)
  - 後方互換の機能追加: マイナー (x.y.0)
  - バグ修正・微調整: パッチ (x.y.z)
- コミットメッセージは**日本語**。バージョン変更時はメッセージに版番号を含めること。
- ステージは変更したファイルのみ (`git add <ファイル>`)。`git add .` / `-A` 禁止。

## 開発メモ
- HTMLシングルファイル構成（`index.html`）
- 外部ライブラリ不使用
- Canvas APIで全画像処理
