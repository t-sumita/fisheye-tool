# fisheye-tool

円周魚眼レンズ変換ツール / Circular Fisheye Lens Converter

## 概要
JPG/PNG画像を球面魚眼レンズ効果で円形に変換するブラウザベースのツール。
インストール不要。fisheye-tool.html をブラウザで開くだけで使用できる。

## 使い方
1. `fisheye-tool.html` をブラウザで開く
2. 画像をドロップ・クリック・Ctrl+Vで入力
3. 各スライダーで効果を調整
4. コピーまたはダウンロードで出力

## 機能
- 球面魚眼レンズ変換
- 湾曲率調整
- 拡大率調整
- 位置補正（X/Y）
- 背景透明化（しきい値・線色指定）
- JP/EN 言語切替
- PNG出力（透明背景対応）

## 技術仕様
- HTMLシングルファイル（外部ライブラリ不使用）
- Canvas API による画像処理
- 対応画像形式: JPG, PNG

### バージョン管理ルール
- 機能追加時はバージョンを更新すること。
  - 破壊的変更・大機能: メジャー (x.0.0)
  - 後方互換の機能追加: マイナー (x.y.0)
  - バグ修正: パッチ (x.y.z)
- バージョン更新時はコミットメッセージにバージョン番号を含めること
